# 사라지는 메모장 — PRD

## 1. 서비스 개요
"사라지는 메모장"은 사용자가 적은 글이 5분 뒤 자동으로 사라지는, 마음 비우기 전용 웹 도구다. 새벽 3시 꼬리 무는 생각이나 하루 끝 잡념을 쏟아낸 뒤 "놓아주는" 경험을 제공한다. 저장도, 로그인도 없다. 오직 지금 이 순간의 생각을 흘려보내는 것이 목적이다.

## 2. 폴더 구조
이 레포는 5개 팀(파트)에 따른 5개 영역으로 나뉜다:
- planning/ — 기획 (무엇을 만들 것인가)
- design/ — 디자인 (어떻게 보여줄 것인가)
- cx/ — CX (누가 어떻게 사용할 것인가)
- marketing/ — 마케팅 (어떻게 알릴 것인가)
- strategy/ — 전략 (어떻게 지속 가능하게 만들 것인가)

각 영역의 상세는 해당 폴더에서 작성된다.

## 3. 기획 (Planning)
- 메모 작성 (간단한 텍스트 입력)
- 5분 타이머 자동 카운트다운
- 잉크가 번지듯 사라지는 애니메이션
- 로그인 없는 익명 사용
- 한 문장 명상적 안내

MVP 범위: 작성 → 5분 타이머 → 자동 삭제, 1개 화면 기준.

상세는 `planning/` 폴더 참조. [작성 예정]

## 4. 디자인 (Design)
전체 비주얼 컨셉은 "명상적 · 치유적"으로, 차분한 다크 톤과 느린 인터랙션이 중심이다.
주요 화면 흐름: 메인(빈 종이) → 작성 중(카운트다운 표시) → 사라짐(번짐 애니메이션) → 초기 상태.

상세는 `design/` 폴더 참조. [작성 예정]

## 5. CX (Customer Experience)
타겟 페르소나: 새벽 3시 잠 못 드는 직장인 — 머릿속 생각을 어디다 쏟을지 모르는 사람.
사용자 여정의 핵심 단계: 진입 → 즉시 작성 → 5분 기다리거나 떠나기 → 다시 찾을 때 아무 것도 남지 않음을 체감.

상세는 `cx/` 폴더 참조. [작성 예정]

## 6. 마케팅 (Marketing)
첫 사용자 확보는 "새벽의 불면러" 타겟에 맞춘 감성 카피로 접근한다. 주요 채널은 인스타그램(감성 포스팅)과 트위터/X(새벽 불면 관련 공감 트윗).

상세는 `marketing/` 폴더 참조. [작성 예정]

## 7. 전략 (Strategy)
무료 기본형 + 선택적 유료 기능(타이머 길이 커스텀, 향 있는 사운드 등)을 검토한다. 지속 가능성은 감성 도구로서의 입소문과 반복 사용에 달려 있다.

상세는 `strategy/` 폴더 참조. [작성 예정]

## 8. 역할 배정 (가상)
이 프로젝트를 팀으로 한다면:

| 팀 | 담당 영역 | 작업 폴더 |
|----|---------|----------|
| PM (본인) | 전체 기획·의사결정 | 루트 |
| 기획 팀 | 기능 정의, MVP 범위 | planning/ |
| 디자인 팀 | UI/UX, 비주얼 | design/ |
| CX 팀 | 페르소나, 여정맵, FAQ | cx/ |
| 마케팅 팀 | 런칭, 채널 | marketing/ |
| 전략 팀 | 수익 모델, 성장 | strategy/ |
