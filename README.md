# 사라지는 메모장

적고 5분 뒤 사라지는, 마음 비우기 전용 웹서비스.

## 프로젝트 정보
- **유형**: 도구 (웹 서비스)
- **타겟 사용자**: 새벽 3시 잠 못 드는 직장인 — 머릿속에 꼬리 무는 생각을 쏟아내고 싶은 사람
- **출시 목표**: [작성 예정]

## 핵심 기능
- 메모 작성 (간단한 텍스트 입력)
- 5분 타이머 자동 카운트다운
- 잉크가 번지듯 서서히 사라지는 애니메이션
- 로그인 없이 익명으로 사용
- 한 문장 명상적 안내 ("이 생각, 5분만 머물러 있을게요")

## 폴더 구조 (5개 팀)
- `planning/` — 기획 팀: 무엇을 만들 것인가
- `design/` — 디자인 팀: 어떻게 보여줄 것인가
- `cx/` — CX 팀: 누가 어떻게 사용할 것인가
- `marketing/` — 마케팅 팀: 어떻게 알릴 것인가
- `strategy/` — 전략 팀: 어떻게 지속 가능하게 만들 것인가
