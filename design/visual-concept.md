## 1. 무드 (Mood)

- **키워드**: 명상적 · 치유적 · 새벽 · 잉크 · 고요
- **레퍼런스 감정**: 조용한 해변, 안개 낀 새벽, 편지지, 낡은 만년필
- **피해야 할 것**: 밝음, 경쾌함, 게임적 요소, 과한 그라디언트, SaaS 느낌

## 4. 화면 구조 레이아웃 (모바일 우선)

```
┌─────────────────────┐
│                     │
│   ⏱  5분  ◀─────▶   │  ← 상단 60px 영역 (고정)
│                     │
│  ─────────────────  │
│                     │
│   │                 │
│   │ (커서 여기)      │
│   │                 │  ← 중앙 캔버스 (가변)
│                     │
│                     │
│   ─────────────     │  ← 하단 안내 (선택적)
│   • • •             │
└─────────────────────┘
```

- 상단 영역: 60px, 타이머 슬라이더
- 캔버스: 가용 높이 전부, padding 24px
- 하단 안내: 기본 숨김, 타이핑 전에만 잠깐 노출

## 5. 잉크 번짐 애니메이션 (방향성)

- **지속**: 2.5초
- **단계**:
  1. 0.0s ~ 0.8s: 글자 색이 서서히 흐려지기 시작
  2. 0.5s ~ 1.5s: 글자 주변에 blur 퍼짐 (마치 물에 젖은 잉크)
  3. 1.0s ~ 2.5s: 전체 opacity가 0으로 수렴
- **CSS 힌트**: `filter: blur(0px → 12px)` + `opacity: 1 → 0` + `transform` 미묘한 움직임
- **`prefers-reduced-motion`**: 단순 500ms 페이드아웃으로 대체
